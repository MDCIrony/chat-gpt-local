CARACTERÍSTICAS PRINCIPALES

Integración con OpenAI

Conexión directa con la API de OpenAI para obtener respuestas inteligentes utilizando los modelos más avanzados.

Soporte Markdown

Renderiza perfectamente contenido en formato Markdown, incluyendo bloques de código, tablas y fórmulas.

Interfaz Cyberpunk

Disfruta de una experiencia visual única con efectos glitch, escaneo y una paleta de colores neón que te sumerge en un ambiente futurista.

Rendimiento Óptimo

Desarrollado con Python 3.12 y Qt 6 para ofrecer una experiencia fluida y rápida en cualquier sistema compatible.

100% Local

La interfaz de usuario funciona completamente en tu equipo, solo las consultas de IA se envían a los servidores de OpenAI.

Personalizable

Código abierto y fácilmente modificable para ajustar los estilos y funcionalidades según tus necesidades.

INSTALACIÓN SENCILLA

1. Clona el repositorio

git clone https://github.com/tuusuario/chat-gpt-local.git

2. Instala las dependencias

cd chat-gpt-local
pip install -e .

3. Configura tu API Key

# Crea un archivo .env con tu clave API
echo "OPENAI_API_KEY=tu_clave_api_aqui" > .env

4. Ejecuta la aplicación

chat-gpt-local
